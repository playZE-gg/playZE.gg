This section contains player-written guides for CS2 Zombie Escape maps. Each guide covers the map's stages, recommended holds, item usage, and common places teams wipe.

## How guides are organized

Each map gets its own page. If a map has a CS2 `_p` port variant (a separately rebuilt version), that variant gets its own page as well — they are treated as independent guides because the layouts, timing, and items often differ.

Example: `ze_best_korea` and `ze_best_korea_p` are separate pages with separate guides.

## Difficulty ratings

| Rating | Description |
|---|---|
| Easy | Beginner-friendly; forgiving holds, slow zombies, clear callouts |
| Medium | Some mechanical skill required; tighter timings |
| Hard | Coordination-dependent; requires experienced leaders and item users |
| Insane | Expert-only; very tight execution margins |

## Contributing a map guide

Use the [map template](https://github.com/playZE-gg/playZE.gg/blob/main/src/content/docs/maps/_map-template.md) as your starting point. Copy it, rename it to the map's filename (e.g. `ze_best_korea.md`), and fill in each section. See [How to Contribute](/contribute/how-to-contribute/) for the PR process.

## Finding guides

Browse the sidebar or use the search bar (top right) to find a specific map.
