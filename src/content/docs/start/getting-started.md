Zombie Escape is a community-made game mode most often found in the Counter-Strike series. This page covers what you need installed, where to find servers, and what to expect before your first round.

## What is Zombie Escape?

A team-based survival game mode where humans must move through a map and complete objectives before zombies infect them all. One or more humans start as zombies; anyone they attack becomes one too. The round ends when humans reach the escape point, or when every human is infected.

:::note
The primary focus of this site is zombie escape(ZE) on Counter-Strike 2. While ZE can be played on other games like Counter-Strike:Source or Team Fortress 2, they should not be considered the default when reading through these pages. When content is specifically for one of these other games, that will be made clear via the title, description or a callout in the page like what you are reading now. 
:::
## What you need

- Counter-Strike 2 installed, it's FREE! (or whichever game you are trying to play on, if supported)
- No mods or plugins required on your end — everything runs server-side

## Finding a server

Use the community server browser in CS2:
1. Open CS2 → Play → Community Server Browser
2. Search for "Zombie Escape" or "ZE" (or search by map name for "ze_")
3. Sort by players, active servers will have 20–64 players

Popular server regions: EU, NA, Asian servers often run different map rotations. Start with whatever is closest to you geographically for best ping.

## First-time settings

Before you join, a few settings make ZE much more playable:

### Volume
ZE relies on audio cues: player voice-chat, zombie growls, item sounds, end-of-stage music. Turn your game volume up, but once in the server, ask how you can adjust these levels individually as the method/commands might be different per server.

### Crosshair and sensitivity
ZE is not a competitive mode (usually). Use whatever settings you are comfortable with.

### Console
Enable the developer console in CS2 settings (under Game). The console is used for certain map visual settings, item commands, and setting custom binds to buy weapons, place pings, or other custom messages.
